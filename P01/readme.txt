Para la correcta ejecución del programa se necesita tener instalados los
paquetes mencionados en 'requirements.txt' y contar con una versión de 
python 3.8 o superior.

La manera de ejecutar el programa es la siguiente:
python gui.py

NOTA: Para el 'filtro' de agregar una marca de agua, es necesario ingresar una
    palabra por la línea de comandos, este va a ser el texto que tenga la 
    marca de agua.
